    About MOES

    MOES, the Multiplayer Online Emulator Suite, is a project that strives to provide functional and accurate web based emulation for a variety of single and multiplayer consoles. Currently, MOES supports GameBoy and NES ROMs, but future plans include adding support for additional consoles, with emulation for Nintendo 64 and GameBoy Advance Consoles in progress. Features such as Debuggers, rom disassembly custom keymapping, and support for custom palettes and tilesets will also be added. If you experience any bugs during gameplay be sure to report them so the issue can be fixed. You can do this by clicking the link at the bottom of the page and sending the devolpers an email or by filing an issue on the MOES github page.

    Features

    Here are some of the key features of MOES

    GameBoy

    • 100% accurate Z80 emulation
    • MBC 0 and 1 suppport

    NES

    • 100% accurate 6502 emulation
    • mapper 0,1,4 support

    In Progress

    • GBA support
    • N64 support
    • Multiplayer server
    • Custom graphics
    • Debuggers
